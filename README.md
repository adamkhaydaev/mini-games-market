# 🎮 MarketGame's

Сайт-витрина игрового магазина. Каталог игр, личный кабинет, поиск,
страницы с описаниями и админ-панель для управления товарами.

Проект полностью на **HTML + CSS + JavaScript**, без бэкенда —
все данные (пользователи, игры, корзина, тема оформления) хранятся
в `localStorage` браузера.

## ✨ Возможности

- 🏠 Главная страница с витриной игр
- 🗂️ Каталог и личный кабинет со списком игр
- 🔎 Живой поиск по играм
- 📄 15 отдельных страниц с описанием игр (RDR2, Counter-Strike 2,
  Elden Ring, Phasmophobia, Among Us, Terraria, Garry's Mod и др.)
- 👤 Регистрация, вход и смена пароля
- 🛠️ Админ-панель: добавление и удаление игр
- 🌗 Переключение тёмной/светлой темы
- 📑 Страницы: «О нас», «Контакты», «Поддержка», «Политика конфиденциальности»

## 🚀 Запуск

Установка не требуется. Достаточно открыть сайт в браузере:

1. Скачать репозиторий (`Code → Download ZIP`) или клонировать:
```bash
   git clone https://github.com/adamkhaydaev/mini-games-market.git
```
2. Открыть файл `Adam.html` двойным кликом.

> Стартовая страница — `Adam.html`.

## 🗂️ Структура проекта

| Файл | Назначение |
|------|------------|
| `Adam.html` | Главная страница |
| `catalog.html` | Каталог игр |
| `dashboard.html` | Личный кабинет / витрина магазина |
| `search.html` | Поиск по играм |
| `description*.html` | Страницы с описанием отдельных игр |
| `login.html` / `register.html` / `change-password.html` | Авторизация |
| `admin.html` | Админ-панель (управление играми) |
| `about.html` / `contacts.html` / `support.html` / `privacy.html` | Информационные страницы |
| `*.jpg`, `*.png` | Изображения и постеры игр |

## 🛠️ Технологии

- HTML5
- CSS3
- JavaScript (Vanilla)
- `localStorage` для хранения данных
- Google Fonts (Orbitron)

## ⚠️ Примечания

- Это учебный/демонстрационный проект. Авторизация и хранение данных
  реализованы на стороне браузера и **не предназначены для реальной
  защиты данных** (пароли хранятся в `localStorage` в открытом виде).
- Данные сохраняются только в текущем браузере и сбрасываются при
  очистке `localStorage`.
